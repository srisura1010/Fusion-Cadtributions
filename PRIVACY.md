# Privacy Policy

**Last updated: July 2026**

CADtributions is a Fusion 360 add-in that tracks your save history and displays it as a contribution graph. This document explains what data the add-in handles.

## What data is collected

None. CADtributions does not collect, transmit, or share any data with the developer, Autodesk, or any third party.

## What data is stored, and where

The add-in stores information locally on your own computer, in a plain JSON file next to the add-in's installation folder. This includes:

- File names, project names, and folder paths for designs you save in Fusion 360
- Version numbers and timestamps of your saves
- Descriptions you enter in Fusion's save dialog

This data never leaves your machine. There are no analytics, no telemetry, and no network requests of any kind built into the add-in.

## Third parties

CADtributions does not use any third party services, libraries, or SDKs that collect data. It does not integrate with any analytics or advertising platform.

## Data retention and deletion

Since all data is stored locally in a file on your own computer, you control it directly. To delete your data, delete the `cadtributions_data.json` file found in the add-in's installation folder, or uninstall the add-in entirely.

## Your control over this data

Because nothing is sent anywhere, there is nothing to request deletion of or revoke consent for beyond what's described above. You can inspect, edit, or delete the local data file at any time.

## Changes to this policy

If this policy changes in the future, the updated version will be posted here with a new "last updated" date.
